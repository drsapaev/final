# Firebase Cloud Messaging (FCM) для push уведомлений
# Инструкция по получению ключа:
# 1. Перейдите в Firebase Console: https://console.firebase.google.com/
# 2. Создайте проект или выберите существующий
# 3. Перейдите в Project Settings (⚙️) > Cloud Messaging
# 4. В разделе "Cloud Messaging API (Legacy)" найдите "Server key"
# 5. Скопируйте Server key и вставьте ниже
# 
# Альтернативный способ (для новых проектов):
# 1. В Firebase Console: Project Settings > Service accounts
# 2. Нажмите "Generate new private key" для создания service account
# 3. Используйте этот JSON файл для аутентификации
# 
# FCM_SERVER_KEY - обязательный для работы push уведомлений
