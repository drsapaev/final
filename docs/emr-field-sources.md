## 📌 Поведение AI с зависимостями

### Осмотр / Лечение
- Требуют заполненных **жалоб**
- Без жалоб → показать: `💡 Для AI-подсказок сначала заполните поле «Жалобы»`
- Никогда не молчать

### Диагноз (ICD-10)
- Работает **независимо**
- Анализирует симптомы и предполагаемый диагноз
- Возвращает структурированный JSON `[{code, label, confidence}]`

## 🚫 Антипаттерны

| ❌ Плохо | ✅ Хорошо |
|----------|-----------|
| Кнопка есть, но ничего не делает | Убрать кнопку или показать объяснение |
| 📜 История выглядит как 🧠 AI | Чёткие иконки и подписи |
| AI молчит при ошибке | Всегда показать фидбек |
| AI для всех полей | AI только там, где нужна генерация |
