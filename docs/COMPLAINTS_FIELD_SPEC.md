# Эталонное поле: «Жалобы»

## Цель документа

Создать **образец идеального UX** для одного поля, который станет эталоном для всех остальных полей в EMRContainerV2.

---

## Почему именно «Жалобы»?

1. **Первое поле** — задаёт тон всему приёму
2. **Свободный текст** — максимум вариаций ввода
3. **AI-релевантно** — здесь AI может реально помочь
4. **Клинически важно** — это начало диагностической цепочки

---

## Состояния поля

### 1. ПУСТО (Empty)

```
┌─────────────────────────────────────────────────────┐
│ ЖАЛОБЫ                                              │
├─────────────────────────────────────────────────────┤
│ ┌─────────────────────────────────────────────────┐ │
│ │ Введите жалобы пациента...                     │ │
│ │ |                                              │ │ ← курсор мигает
│ │                                                 │ │
│ └─────────────────────────────────────────────────┘ │
└─────────────────────────────────────────────────────┘
```

**Поведение:**
- Placeholder серый, не мешает
- Auto-focus при открытии EMR (курсор уже здесь)
- Border нейтральный (#e5e5e5)
- Высота: 80px минимум

**Клавиатура:**
- Любая буква → начало ввода (состояние 2)
- Tab → переход к следующему полю
- Shift+Tab → переход назад (header/buttons)

**Мышь:**
- Клик → фокус (если не было)

```
├─────────────────────────────────────────────────────┤
│ ┌─────────────────────────────────────────────────┐ │

│ │                                                 │ │
│ │                                                 │ │

│   💾 Auto-save через 3 сек                          │ ← subtle hint
└─────────────────────────────────────────────────────┘
```

**Поведение:**
- Border синий (#007aff) — фокус

- Textarea растягивается при необходимости

**Auto-save:**
- Debounce 3 секунды после последнего нажатия
- Subtle индикатор "Сохранено" (не блокирует)

**Клавиатура:**
- Enter → новая строка (это textarea)
- Tab → переход к следующему полю AND сохранение
- Escape → ничего (не закрываем)

**Мышь:**
- Клик вне поля → blur, сохранение

### 3. С AI-ПОДСКАЗКОЙ (AI Suggestion)

```
┌─────────────────────────────────────────────────────┐
│ ЖАЛОБЫ                                              │
├─────────────────────────────────────────────────────┤
│ ┌─────────────────────────────────────────────────┐ │
│ │ Головная боль в затылке                        │ │
│ │                                                 │ │
│ └─────────────────────────────────────────────────┘ │
│                                                      │
│ ┌─ 💡 ─────────────────────────────────────────────┐ │
│ │ давящего характера, усиливается к вечеру,      │ │ ← AI предложение
│ │ сопровождается шумом в ушах                    │ │
│ │                          [✓ Принять] Alt+Enter │ │
│ └─────────────────────────────────────────────────┘ │
└─────────────────────────────────────────────────────┘
```

**Когда появляется:**
- После паузы ввода 1.5 сек
- Только если есть релевантный контекст
- Только если врач не отключил AI

**Поведение:**
- Появляется плавно (fade in 200ms)
- Не закрывает поле ввода
- Не мешает продолжать печатать

**Клавиатура:**
- Alt+Enter → принять подсказку (append к тексту)
- Escape → скрыть подсказку
- Продолжение печати → подсказка обновляется или исчезает

**Мышь:**
- Клик на [Принять] → append текст
- Клик вне → скрыть подсказку

**Важно:**
- AI **дополняет**, а не заменяет
- Подсказка insert'ится с ", " если текст не пустой
- После принятия — фокус остаётся в поле

## Спецификация компонента

```typescript
interface ComplaintsFieldProps {
  // Data
  value: string;
  onChange: (value: string) => void;
  
  // State
  isEditable: boolean;        // false после подписи
  isFocused?: boolean;        // контролируемый фокус
  
  // AI
  aiSuggestion?: string;      // текущая подсказка
  onAcceptAI?: () => void;    // принять подсказку
  onDismissAI?: () => void;   // скрыть подсказку
  
  // Validation
  error?: string;             // сообщение ошибки
  
  // UX
  autoFocus?: boolean;        // фокус при mount
  onFieldTouch?: () => void;  // для telemetry
}
```

## Поведение AI-подсказки

### Триггер появления:
1. Врач вводит текст
2. Пауза ≥ 1.5 секунды
3. Текст ≥ 10 символов
4. AI включён в настройках

### Формат подсказки:
- Дополнение к существующему тексту
- Начинается с lowercase (будет append через ", ")
- Не повторяет то, что уже написано

### Принятие подсказки:
```javascript
const acceptSuggestion = () => {
  if (!aiSuggestion) return;
  
  const separator = value.trim().endsWith('.') ? ' ' : ', ';
  const newValue = value.trim() + separator + aiSuggestion;
  
  onChange(newValue);
  onDismissAI();
  // Фокус остаётся в поле, курсор в конец
};
```

## Валидация

### При сохранении черновика:
- Нет валидации
- Всё сохраняется

### При подписании:
```javascript
const validateForSigning = (emr) => {
  const errors = [];
  
  if (!emr.complaints?.trim()) {
    errors.push({ field: 'complaints', message: 'Жалобы обязательны' });
  }
  
  if (!emr.diagnosis?.trim()) {
    errors.push({ field: 'diagnosis', message: 'Диагноз обязателен' });
  }
  
  return errors;
};
```

## Keyboard Navigation Map

| Состояние | Key | Действие |
|-----------|-----|----------|
| Пусто | Any letter | Начать ввод |
| Пусто | Tab | Следующее поле |
| Ввод | Tab | Сохранить + следующее поле |
| Ввод | Shift+Tab | Сохранить + предыдущее поле |
| Ввод | Enter | Новая строка |
| Ввод | Escape | Ничего |
| AI показан | Alt+Enter | Принять подсказку |
| AI показан | Escape | Скрыть подсказку |
| AI показан | Любая буква | Продолжить ввод, AI обновляется |
| Подписано | Tab | Пропустить (не focusable) |
