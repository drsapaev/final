## Выявленные UX-требования

### ✅ ОБЯЗАТЕЛЬНО (MVP)

1. **Auto-focus** в поле "Жалобы" при открытии EMR
2. **Tab-навигация** между всеми полями и секциями
3. **Enter** для подтверждения AI-подсказок
4. **Свёрнутые секции** пропускаются при Tab
5. **Витальные из истории** — подтягивать рост/вес
6. **ИМТ авто-расчёт**
7. **МКБ-10 dropdown** с поиском
8. **Статус визита** меняется мгновенно

### ⚠️ ВАЖНО (Этап 1)

1. **AI-подсказки с задержкой 1.5 сек** — не раньше
2. **Предупреждение при высоком АД** — subtle, не блокирующее
3. **Шаблоны назначений** для частых случаев
4. **Подтверждение подписи** — диалог или двойной клик

### 🔮 ПОЗЖЕ (Этап 2+)

1. Голосовой ввод
2. История назначений пациента в sidebar
3. Проверка drug-drug interactions
