# ЭМК v2 — Краткая справка

## Статусы сохранения

| Индикатор | Что это значит |
|-----------|----------------|
| ● Есть изменения | Ждите, идёт автосохранение |
| ✓ Сохранено | Всё в безопасности |
| 💾 Сохранение... | Прямо сейчас сохраняется |
| ⚠️ Конфликт | Выберите действие |

---

## Кнопки

| Кнопка | Действие |
|--------|----------|
| 💾 Сохранить | Сохранить вручную |
| ✅ Подписать | Завершить и подписать документ |
| 📝 Поправка | Изменить подписанный документ |
| ↩️ Отменить | Отменить последнее изменение |
| 📜 История | Посмотреть все версии |

---

## Горячие клавиши

| Клавиша | Действие |
|---------|----------|
| `Ctrl+S` | Сохранить |
| `Ctrl+Z` | Отменить |
| `Ctrl+Shift+Z` | Повторить |

---

## При конфликте

1. **Сравнить** — посмотреть различия
2. **Загрузить** — взять версию с сервера
3. **Поправка** — сохранить как поправку

---

## Помните

✅ Дождитесь «Сохранено» перед закрытием  
✅ После подписи — только поправки  
✅ AI предлагает, но решаете вы
