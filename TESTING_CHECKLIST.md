## 🚀 Быстрый старт

### 1. Откройте браузер
- URL: `http://localhost:5173`
- Откройте DevTools (F12) → вкладка **Console**

### 2. Войдите в систему
- Используйте учетную запись врача (cardio/derma/dentist) или лаборанта

### 3. Начните тестирование

**Порядок тестирования:**

1. **Панель Кардиолога** → `http://localhost:5173/cardiologist?tab=queue`
2. **Панель Дерматолога** → `http://localhost:5173/dermatologist?tab=queue`
3. **Панель Стоматолога** → `http://localhost:5173/dentist?tab=queue`
4. **Панель Лаборатории** → `http://localhost:5173/lab-panel?tab=queue`

## 🔧 Полезные команды

### Проверка готовности системы
```bash
python check_testing_readiness.py
```

### Запуск backend (если не запущен)
```bash
cd backend
python -m uvicorn app.main:app --reload --host 0.0.0.0 --port 18000
```

### Запуск frontend (если не запущен)
```bash
cd frontend
npm run dev
```
